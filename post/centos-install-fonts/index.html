CentOS 下安装字体

2020-02-17 · 1 min · ovenx
Table of Contents
  • 查看字体
  • 安装字体
  • 建立字体缓存

本文的系统版本 centos7 x64

查看字体#

fc-list

安装字体#

cp xxx.ttf /usr/share/fonts/ # copy fonts to /usr/share/fonts

建立字体缓存#

cd /usr/share/fonts/
chmod 755 *.ttf # 修改字体权限
mkfontscale # 依赖 mkfonts
mkfontdir
fc-cache -fv # 依赖 fontconfig
