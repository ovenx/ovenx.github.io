## 开启 Adminstrator 账户

选择 `计算机` -> `管理` ->`本地用户和组` -> `用户`

找到 Administrator, 去除 `此账户已禁用` 勾选

## 获取超级管理员权限

打开策略编辑器

```bash
win + r
gpedit.msc
```

选择 `计算机配置` -> `Windows 设置` -> `安全策略` -> `本地策略` -> `安全选项`

找到下面两项，设置为 `禁止`

* 用户账户控制：以管理员批准模式运行所有管理员
* 用户账户控制：用于内置管理员账户的管理员批准模式

重新启动后 administator 账户即可获得最高权限

> 此方法只适用于 win10 专业版/教育版/企业版
